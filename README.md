# Colloid_Exam

Конспекты по всем 61 билетам экзамена по коллоидной химии, на основе учебника Щукина «Коллоидная химия».

## Как открыть в Obsidian

1. Открой Obsidian.
2. На стартовом экране (или через меню слева внизу, шестеренка-вход) выбери "Open folder as vault".
3. Укажи папку `Colloid_Exam` (эту самую).
4. Готово. Все билеты лежат в папке `Билеты_md`, открывай нужный файл по номеру.

Ссылки вида `[[билет_05]]` внутри текста кликабельные, по ним можно переходить между билетами. Картинки подгружаются автоматически из папки `Картинки`, отдельно их открывать не нужно.

Если формулы (LaTeX, в `$...$`) не отображаются красиво, проверь в настройках Obsidian, что включен рендеринг математики (обычно работает из коробки).

## Что в папке

- **`Билеты_md/`** — главное. 61 файл `билет_01.md` ... `билет_61.md`, по одному на каждый билет. Здесь весь разбор: определения, формулы, примеры, картинки, ссылки на другие билеты. Используй эти файлы для подготовки.

- **`Картинки/`** — все иллюстрации (схемы, графики из учебника), которые вставлены в файлы билетов. Открывать отдельно не нужно, они и так показываются внутри `.md` файлов.

- **`Билеты_pdf/`** — те же билеты, но в формате PDF (на случай если нужно распечатать или открыть не в Obsidian).

- **`colloid-exam-2026 (1).pdf`** — список билетов с экзамена, исходный документ с формулировками тем.

- **`Щукин. Коллоидная химия.pdf`** — учебник, по которому написаны все конспекты. Если в каком-то билете нужно посмотреть подробности или оригинальный текст, можно свериться с ним.

- **`_map_билеты_учебник.md`** — служебный файл, таблица соответствия "билет → раздел учебника со страницами". Полезен, если хочешь найти, где именно в учебнике искать тему билета.

- **`.git`** — служебная папка для версионирования (история изменений файлов). Трогать не нужно.
